Function that creates a copy of a CSV file in different location while verifying their identity from the register system view. The identity of src and dest changes the function behavior if the src file was deleted during the program run.

Parameters
srcPath to the file which is to be copied.
destPath to the file into which the src is to be copied.
roomfilePath to the file containing room data. Its purpose is to verify whether src or dest is this file.
roombackupPath to the file containing backup of room data. Its purpose is to verify whether src or dest is this file.
regfilePath to the file containing registration data. Its purpose is to verify whether src or dest is this file.
regbackupPath to the file containing backup registration data. Its purpose is to verify whether src or dest is this file.
