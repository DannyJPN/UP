Function is an analogy to function strcmp. Compares two dates.

Parameters
date1First date to be compared.
date2Second date to be compared.
Returns
This function is analogic to strcmp,thus it returns -1 if first date is lower (=earlier),1,if the first date is higher (=later) or 0 if the dates are equal.
